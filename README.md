q3dpiechart
===========

A 3D pie chart library using qt.

    q3dpiechart *piechart = new q3dpiechart(this);
    piechart->addPie(10, "aaa");
    piechart->addPie(13, "bbb");
    piechart->addPie(11, "ccc");
